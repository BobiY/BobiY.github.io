这篇文章主要说明的是，将项目中引入的 css 文件单独打包的 webpack 插件

extract-text-webpack-plugin

这个插件就是将 css 单独打包的关键所在，下面来介绍它的用法,首先是安装，使其成为你的项目依赖。

npm install extract-text-webpack-plugin -D

然后我们要将其加入到我们的 webpack 的配置文件中去

{% highlight javascript %} //在页头引入 const ExtractTextWebpackPlugin = require("extract-text-webpack-plugin"); // 然后是你的 css loader 下面的是 webpack 2.0 以后的写法 { test:/\.css$/, use:ExtractTextWebpackPlugin.extract({ fallback:"style-loader", use:"css-loader" }) } // 然后在 pulgins 字段中添加插件 style.css 就是你打包后的文件名，可以自定义 pulgins:[ new ExtractTextWebpackPlugin("style.css"); ] {% endhighlight %}
